# InputGenerator

En este repositorio tenemos un conjunto de codigos para generar inputs aleatorios o a su convenir para distintos ejemplos, como para Medicina, Bioinformatica, modelos de gases de efecto invernadero o como para el problema matematico MDO que compromete una bacteria y un conjunto de variables
